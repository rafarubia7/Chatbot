Informações da Aplicação

Sobre o Projeto

O Chatbot Inteligente SENAI é uma aplicação web desenvolvida para fornecer informações sobre o SENAI São Carlos de forma interativa e automatizada.

O sistema utiliza inteligência artificial para responder perguntas sobre cursos, processos de inscrição, informações institucionais e muito mais.

Funcionalidades Principais

  • Chat em tempo real com IA
  • Informações sobre cursos disponíveis
  • Processos de inscrição e matrícula
  • Dados institucionais do SENAI
  • Sistema de autenticação de usuários
  • Histórico de conversas salvo
  • Interface responsiva e moderna
  • Suporte a múltiplos usuários

Arquitetura do Sistema

Backend: Flask (Python) com SQLAlchemy para banco de dados

Frontend: HTML5, CSS3 e JavaScript

IA: Processamento de linguagem natural (NLTK)

Banco: SQLite para desenvolvimento

Cache: Redis para otimização

Recursos de IA

  • Processamento de linguagem natural
  • Reconhecimento de padrões em perguntas
  • Respostas contextuais inteligentes
  • Aprendizado com interações
  • Busca fuzzy para melhor precisão
  • Análise de intenção do usuário

Segurança

  • Autenticação de usuários
  • Sessões seguras
  • Validação de dados
  • Proteção contra injeção SQL
  • Sanitização de inputs
  • Logs de auditoria

Base de Conhecimento

O chatbot possui uma extensa base de conhecimento sobre:

  • Cursos técnicos e profissionalizantes
  • Processos de inscrição e matrícula
  • Informações sobre salas e laboratórios
  • Dados institucionais do SENAI
  • Perguntas frequentes
  • Contatos e localização
