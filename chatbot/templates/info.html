Informações da Aplicação

Sobre o Projeto

O Cadu é um assistente virtual desenvolvido para auxiliar alunos, funcionários e visitantes do SENAI São Carlos com informações institucionais.

O sistema utiliza integração com LM Studio para processamento de linguagem natural e oferece uma interface web moderna e intuitiva.

Desenvolvido especificamente para a Escola SENAI São Carlos – "Antonio A. Lobbe", o chatbot fornece informações precisas sobre cursos, horários, funcionários, salas e processos administrativos.

Funcionalidades Principais

  • Chat inteligente com processamento de linguagem natural via LM Studio
  • Informações sobre cursos técnicos, superiores e de aprendizagem
  • Consulta de horários de professores, salas e turmas
  • Informações sobre funcionários e setores
  • Processos de inscrição e matrícula
  • Localização de salas e instalações
  • Sistema de autenticação com login e registro
  • Acesso sem login (modo visitante)
  • Histórico de conversas salvo por usuário
  • Gerenciamento de múltiplas conversas
  • Perfil de usuário com upload de avatar
  • Copiar e editar mensagens
  • Interromper geração de respostas
  • Reconhecimento de voz (Web Speech API)
  • Sistema de cache inteligente
  • Interface responsiva e moderna

Arquitetura do Sistema

Backend: Flask 3.0.2 (Python) com SQLAlchemy ORM

Frontend: HTML5, CSS3 e JavaScript (Vanilla JS)

IA: LM Studio com modelos de linguagem local (Llama 3.2 3B Instruct)

Banco de Dados: MySQL com suporte a usuários anônimos

Cache: Sistema de cache JSON para otimização de respostas

Busca: FuzzyWuzzy para correspondência de texto inteligente

Autenticação: Sistema de sessões Flask com suporte a usuários anônimos

Recursos de IA e Processamento

  • Processamento de linguagem natural via LM Studio
  • Respostas contextuais baseadas em histórico de conversa
  • Detecção de mensagens sem sentido (gibberish)
  • Busca fuzzy (FuzzyWuzzy) para correspondência de texto
  • Análise de intenção e classificação de escopo
  • Detecção de small talk (cumprimentos, despedidas)
  • Processamento especializado para horários e localizações
  • Personalização de respostas com nome do usuário
  • Sistema de cache para respostas frequentes
  • Fallback inteligente quando LM Studio não está disponível

Segurança e Privacidade

  • Sistema de autenticação com sessões Flask
  • Isolamento de dados por usuário (chats privados)
  • Suporte a usuários anônimos com sessões temporárias
  • Validação de dados com SQLAlchemy ORM
  • Proteção contra injeção SQL (ORM)
  • Sanitização de HTML nas mensagens
  • Gerenciamento seguro de senhas
  • Chaves estrangeiras com CASCADE para integridade

Base de Conhecimento

O chatbot possui uma extensa base de conhecimento estruturada sobre o SENAI São Carlos:

  • Cursos: Técnicos, Superiores, Aprendizagem Industrial e Qualificação
  • Horários: Professores, salas e turmas (dados em JSON estruturado)
  • Funcionários: Informações sobre setores e equipe
  • Localização: Salas, laboratórios, biblioteca, refeitório e outras instalações
  • Processos: Inscrição, matrícula e processos administrativos
  • Institucional: Empresas parceiras, eventos e diferenciais
  • Perguntas Frequentes: Bolsas, certificados, estágios e transferências
  • Contatos: Telefones, emails e horários de funcionamento
  • Informações Adicionais: Áreas de atuação, redes sociais e serviços
